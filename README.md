# homebrew-gt-tools

Homebrew tap for [gt-tools](https://github.com/john-farina/gt-tools) — CLI tools for Graphite stacked PRs.

## Install

```bash
brew tap john-farina/gt-tools
brew install gt-drain
```

Installs both `gt-drain` and `gt-rescue`.

## Update

```bash
brew upgrade gt-drain
```
